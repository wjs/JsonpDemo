JsonpDemo
=========

这是一个用 nodejs 来实现的 jsonp 跨域数据请求的小例子。  

#### 使用方法
打开一个终端，执行：  
    
    git clone https://github.com/weijinshi/JsonpDemo.git
    cd JsonpDemo
    npm install express
    node 1.js

打开另一个终端，到 JsonpDemo 目录下，执行：  
    
    node 2.js

打开chrome浏览器，访问 `http://127.0.0.1:3000`，打开开发者工具查看点击对应按钮的效果。
